# Streaming MacroPad

This is a custom MacroPad designed for live streaming.

## Features
- D8 → Mute/Unmute Mic + LEDs Indicator
- D9 → Start/Stop Stream
- D10 → Switch Scene 1
- D11 → Switch Scene 2
- Rotary → Volume Up/Down
- LEDs → Mic status

## Files
- CAD/ → 3D model of the case
- PCB/ → PCB files (KiCAD)
- Firmware/ → Python KMK code

## BOM
- Seeed XIAO RP2040
- Rotary Encoder
- 4 Push Buttons
- 2 LEDs
- Resistors, wires, etc.
